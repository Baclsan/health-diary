# Health Diary (MVP)

Локальный мобильный MVP приложения «Дневник здоровья» на React + Vite + TypeScript.

## Что реализовано

- Главная страница с приветствием, hero-блоком, последними записями и статистикой за 30 дней.
- Экран добавления записи о головной боли.
- Экран истории с удалением записей.
- Хранение данных в `localStorage` без backend.
- Русскоязычный интерфейс и дисклеймер о медицинских ограничениях.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
